# To Do List
A very basic JavaScript practice project. Create a very basic to-do list and mark off items as you complete them.

## Table of Contents
- [Features](#features)
- [Usage](#usage)
- [Tech Stack](#tech-stack)
- [Roadmap](#roadmap)

## Features
- Add Tasks
- Complete Tasks
- Delete tasks

## Usage
  - Type the title of task into the input field and hit "enter" or click "Add Task".
  - Click "Delete" to delete a task.
  - Mark a task as finished by clicking "Complete"

## Tech Stack

### Core
- **JavaScript (ES6+)**
